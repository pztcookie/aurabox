AuraBox

Emotion check-in → aura selfie → kawaii rubber-hose food creature lab. Run the small local server, then open the app in your browser.

API usage & daily limits The default image backend uses a shared free service (Pollinations). It is meant for demos, learning, and light personal use. Requests are limited per day and can slow down or fail if too many people use it at once. Please don’t spam clicks — be kind to shared infrastructure. For heavy use, switch to a paid API or self-hosted model (see README.md).
How to run
  1. Open a terminal in the aurabox folder.
  2. Run: 
    python3 server.py
  3. Click the button below (or open http://127.0.0.1:8000/index.html).
Open AuraBox & Character Lab

This link only works after server.py is running. Full details: README.md.

Why not double-click index.html?

Character generation talks to /api/flux-generate on the same computer. That requires the Python server — opening the file as file://... won’t load images correctly.
